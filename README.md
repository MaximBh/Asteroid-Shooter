# Asteroid-Shooter

ENG
1. Project Overview

1.1. The project is a two-dimensional arcade game in the space shooter genre.

1.2. The player controls a spaceship in open outer space and must destroy asteroids while avoiding collisions with them.

2. Project Functionality

2.1. Spaceship Control

Rotation of the spaceship to the left and right

Acceleration of the spaceship in the direction it is facing

Firing projectiles

2.2 Asteroids

Random spawning and movement of asteroids

Several types of asteroids differing in size and speed

Splitting of asteroids into smaller ones when hit

2.3 Projectiles

Projectile movement in the direction of the shot

Limited lifetime of projectiles


3. Gameplay

Gradual increase in difficulty due to an increasing number of asteroids

Scoring points for destroying asteroids

Display of the game state and results on the screen

RU
1. Суть проекта

1.1 Проект представляет собой двумерную аркадную игру в жанре космического шутера

1.2 Игрок управляет космическим кораблём в открытом космическом пространстве и должен уничтожать астероиды, избегая столкновений с ними


2. Функционал проекта

2.1 Управление кораблём

Поворот корабля влево и вправо

Ускорение корабля в направлении его носа

Стрельба снарядами

2.2 Астероиды

Случайное появление и движение астероидов

Несколько типов астероидов, отличающихся размером и скоростью

Разделение астероидов на более мелкие при попадании

2.3 Снаряды

Полёт снарядов в направлении выстрела

Ограниченное время существования снарядов


3. Игровой процесс

Постепенное усложнение за счёт увеличения количества астероидов

Подсчёт очков за уничтоженные астероиды

Отображение состояния игры и результатов на экране
