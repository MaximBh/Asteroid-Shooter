1. Общие сведения
1.1 Назначение проекта

Разработка двумерной аркадной игры с использованием библиотеки Python Arcade, включающей стартовое и финальное окна, систему уровней, подсчёт результатов, визуальные и звуковые эффекты.

1.2 Цель проекта

Создание игрового приложения, демонстрирующего:

работу со спрайтами и анимацией

обработку столкновений

физическую модель движения

систему уровней

хранение игровых данных

2. Функционал
2.1 Окна приложения

Стартовое окно (Main Menu)

Игровое окно

Финальное окно (Game Over)

2.2 Игровые объекты

Космический корабль

Астероиды нескольких типов

Снаряды

Частицы эффектов (взрывы)

2.3 Управление

Поворот корабля

Ускорение

Стрельба

Переход между окнами

3. Игровой процесс
3.1 Уровни

Игра разделена на уровни

С каждым уровнем увеличивается:

количество астероидов

их скорость

3.2 Подсчёт очков

Очки начисляются за уничтожение астероидов

Лучший результат сохраняется в файл

4. Используемые технологии
4.1 Спрайты и анимация

Все игровые объекты реализованы через arcade.Sprite

Анимация реализована через обновление состояний спрайтов

4.2 Столкновения

Используется встроенная система arcade.check_for_collision

4.3 Система частиц

Используются эффекты взрывов

4.4 Звук

Используются звуковые эффекты Arcade

4.5 Физика

Реализована собственная физическая модель:

инерция

ускорение

ограничение скорости

4.7 Хранение данных

Данные сохраняются в файл формата .txt или .csv

Хранится лучший результат игрока
